🎵 API - Sistema de Músicas VIP

Uma API RESTful desenvolvida em Node.js que gerencia a autenticação de usuários e fornece acesso a uma lista exclusiva de músicas.


## 🔗 Links do Projeto
* **Deploy da API:** `https://sistema-de-musica.onrender.com`
* **Documentação Interativa (Postman):** [Clique aqui para acessar a documentação detalhada das rotas](https://documenter.getpostman.com/view/52675959/2sBXwmPCKd)


## 💻 Como testar a aplicação
1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` (dentro da pasta `public`) no seu navegador.
3. Teste qualquer possivel emai e senha, que irá notificar que o mesmo foi cadastrado.
4. O Front-end se conectará automaticamente à API hospedada na nuvem!
